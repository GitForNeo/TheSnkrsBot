# TheSnkrsBot

## Installing:

1. Use "Download Zip" or clone my git repository.
2. Download the latest version from [here](ttp://thesnkrsbot.com/snkrsbot_setup.exe).
3. Replace thesnkrsbot.exe with the one you got from step 1.
4. Enjoy!
